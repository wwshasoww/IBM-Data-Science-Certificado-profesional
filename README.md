# Casos practicos del curso de IBM DATA SCIENCE

En cada uno de los casos practicos se habla del objetivo a cumplir y el fin de cada uno de los proyectos.
Este curso consto de 12 modulos de los cuales se adquieren cualidades propiar de un cientifico de datos como:
    - Uso de las herramientas de cientifico de datos: Python y sus librerias, VSCODE, Jupyter notebook, SQL, bases de datos, Plotly.

Modulos:
  - What is Data Science?
  - Tools for Data Science
  - Data Science Methodology
  - Python for Data Science, AI & Development
  - Python Project for Data Science
  - Databases and SQL for Data Science with Python
  - Data Analysis with Python
  - Data Visualization with Python
  - Machine Learning with Python
  - Applied Data Science Capstone
  - Generative AI: Elevate Your Data Science Career
  - Data Scientist Career Guide and Interview Preparation

Cada uno de estos modulos esta orientado para obtener la base de un cientifico de datos y unas buenas habilidades para el desarrollo de cientifico de datos utilizando casos practicos reales.

SKILLS:
  - Creacion de cuadros de mando
  - Enfoque científico
  - Recopilacion de datos
  - Depuracion de datos
  - Preparacion de los datos
  - Alfabetización informática
  - Evaluacion de modelos
  - Presentacion de datos
  - Visualizacion interactiva de datos
  - Representacion gráfica
  - Bases de datos
  - SQL
  - WEB SCRAPPING
  - Machine learning
  - Analisis estadístico
  - Inferencia
  - Pipelines

La conclusion final es un curso con los cuales adquirí dominio los conocimientos prácticos más actualizados que los científicos de datos utilizan en su trabajo diario, los lenguajes y las bibliotecas que utilizan los científicos de datos profesionales, incluidos Python y SQL, Importe y limpie conjuntos de datos, analice y visualice datos, y construya modelos y canalizaciones de aprendizaje automático y Aplique sus nuevos conocimientos a proyectos del mundo real y cree una cartera de proyectos de datos.
